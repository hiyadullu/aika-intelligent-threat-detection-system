Abstract:

Aika: Intelligent Threat Detection through Cognitive Vision is a pioneering project that seamlessly combines artificial intelligence (AI), computer vision, and robotics to address modern home security challenges. Conceived as a dog-shaped robotic system, Aika is equipped with a webcam, sensors, and an Arduino microcontroller to deliver advanced surveillance and hazard detection. By leveraging state-of-the-art technologies like YOLOv8 for object detection, MediaPipe for hand tracking, and face_recognition for facial identification, the system provides a comprehensive and intelligent approach to ensuring household safety. The core functionality of Aika includes real-time facial recognition to identify authorized individuals, object detection to spot potentially hazardous items such as knives, firearms, or chainsaws, and tracking suspicious hand movements to detect unusual or threatening behavior. The system integrates OpenCV for efficient video processing and NumPy for precise calculations, ensuring robust and accurate analysis of captured visual data. Pre-trained machine learning models and neural networks enable Aika to process complex data quickly, making it capable of distinguishing threats with high confidence. Aika's hardware setup features an Arduino microcontroller to handle input from motion and environmental sensors, as well as a webcam for visual input. The system is designed to be user-friendly, responding to voice commands and providing real-time alerts for enhanced interaction and usability. By employing threshold-based detection for suspicious movements and a predefined confidence score for object recognition, the system ensures reliability and reduces false alarms. This project demonstrates the potential of integrating AI and robotics for smart home security systems. Aika not only serves as a vigilant watchdog but also enhances user convenience by intelligently analyzing its environment and taking proactive measures against threats. As an affordable, scalable, and efficient security solution, Aika sets a benchmark for the next generation of cognitive robotic systems.
